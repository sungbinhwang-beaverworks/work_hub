2 변경사항

1 연동설정 버튼 + 모달 추가

외부연동 현황 테이블 티오더 행에 "연동설정" 버튼을 추가한다.
클릭 시 "T-order 연동설정" 모달이 열린다.

참고 패턴: 식권대장(MealKingSettingModal)
API 키 인증형, 장비 매핑 없음, 심플한 2필드 구조.

2 상품연동 버튼 추가

외부연동 현황 테이블 티오더 행에 "상품연동" 버튼을 추가한다.
클릭 시 현재 매장의 전체 상품/카테고리 정보를 T-order 서버에 전달한다.
모달 없이 바로 동작한다 (기존 etc1='Y' 상품연동 버튼 패턴과 동일).

참고 패턴: 기존 상품연동 버튼 (useMutationStmExtnlItlokProduct)

3 매장 개점/마감 연동 제외

이상환님 요청으로 매장 개점/마감 연동은 제외한다.
API 명세서에도 TBD 상태이다.

마감 상태에서 주문 인입 시 당일자로 자동개점되면서 매출이 적재되며,
이는 서버 로직으로 처리한다.
