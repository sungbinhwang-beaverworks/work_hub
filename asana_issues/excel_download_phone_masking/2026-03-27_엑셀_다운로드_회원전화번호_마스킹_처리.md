# (보강) 매출조회 엑셀 다운로드 시 회원 전화번호 마스킹 처리

|------|------|
| 담당 | 황성빈 |
| 일자 | 26.03.27 |
| 프로덕트 | 매장관리 |
| 출처 | 26.3.25 더본IT기획팀 X 개발본부 주간회의 |

---

## 요청배경

개인정보보호법에 의거, 매장관리에서 회원정보를 화면에 보여주는 것은 문제되지 않으나, 회원정보를 엑셀 다운로드 받는 것은 법적 문제가 된다. 특정 매출조회 화면에서 매출 데이터 엑셀 다운로드 시 회원 전화번호 컬럼이 함께 다운로드되고 있음이 확인되었다.

## 요청사항

회원 전화번호가 포함된 엑셀 다운로드에 마스킹 처리(두 자리 ** 이상)를 적용한다. 마스킹 처리가 어려울 경우 해당 컬럼을 삭제 처리한다.

## 법무 확인

마스킹 처리(두 자리 ** 이상) 시 법적 문제없음 확인 (김지원-김보일)

---

## 대상 화면 (7건)

코드 기반 분석 결과. 엑셀 다운로드 버튼 활성 + 회원 전화번호 컬럼 포함이 확인된 화면.

| # | 메뉴 경로 | 엑셀 내 컬럼명 |
|---|----------|-------------|
| 1 | 매출관리 > 거래내역 > **실시간 결제내역** | 주문자 연락처 |
| 2 | 매출관리 > 거래내역 > **실시간 매출내역** | 주문자 연락처 |
| 3 | 부가서비스 관리 > 포인트·스탬프 서비스 > **포인트회원매출내역** | 연락처 |
| 4 | 부가서비스 관리 > 포인트·스탬프 서비스 > **스탬프회원매출내역** | 연락처 |
| 5 | 브랜드관리 > 브랜드공지·이벤트관리 > **브랜드쿠폰팩관리** (쿠폰조회) | 회원연락처 |
| 6 | 브랜드관리 > 브랜드운영관리 > **브랜드멤버십관리** (등급 상세 모달) | 연락처 |
| 7 | 매장관리 > 매장운영관리 > **매장멤버십관리** (등급 상세 모달) | 연락처 |

※ 코드 분석 기반이며 실제 동작 확인은 개발팀 검증 필요

## 참고

- 기존 마스킹 유틸리티 `phoneFormatForCommonTable(value, true)` 사용 시 마스킹 적용됨 (010-****-5678)
- 알림톡 발송이력 3건은 이미 마스킹 적용 완료
- 포인트/스탬프 회원관리(회원 현황/내역 조회)는 엑셀 다운로드 버튼이 주석 처리되어 현재 미사용
