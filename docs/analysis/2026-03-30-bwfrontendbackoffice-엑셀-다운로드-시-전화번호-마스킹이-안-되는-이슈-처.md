# 분석 보고서: 회원 정보 엑셀 다운로드 시 전화번호 마스킹 미적용 이슈

## 1. 분석 대상

*   **프로젝트**: `bw_frontend_backoffice`
*   **전체 파일 수**: 51개
*   **관련 파일**:
    *   `src/apis/cntr/user/list.ts` (회원 목록 조회 API 정의)
    *   `src/apis/cntr/user/totlist.ts` (회원 상세 정보 조회 API 정의)
    *   `src/apis/Hmac.ts` (API 호출 헬퍼 함수, HMAC 생성)
    *   `src/apis/CommonResponseType.ts` (공통 응답 타입 정의)
    *   `package.json` (프로젝트 의존성 정의, 엑셀 관련 라이브러리 포함 여부 확인)
    *   `src/App.tsx` (메인 애플리케이션 엔트리 포인트, 전역 설정 및 라이브러리 초기화)

## 2. 현황

사용자 요청에 따르면, '회원 정보 엑셀 다운로드' 기능 사용 시 전화번호(`mobileNo`) 필드가 마스킹 처리되지 않고 그대로 노출되는 현상이 확인됩니다. 이는 개인정보 보호 규정 위반 소지가 있는 심각한 버그입니다.
