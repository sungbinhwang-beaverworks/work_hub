## 3. 현재 코드 상태

### 동작하는 것
- **에이전트 코어 파이프라인** — POST /api/pipeline → 관리자(분류) → 분석관(분석) → 기획자(기획) → 완료
- **프로젝트 폴더 접근** — beaver_chat_bot 등 manual_automation/ 하위 코드 읽기
- **에이전트 간 메시지** — handoff, completion_report 전달
- **결과 저장** — docs/analysis/, docs/planning/에 마크다운 파일 자동 생성
- **단일 에이전트 대화** — SSE 스트리밍 채팅
- **웹앱** — PixiJS 2D 오피스 (에셋 미완)

### 아직 안 된 것
- 웹앱 채팅에서 파이프라인 트리거 연결 (채팅 → /api/pipeline 자동 호출)
- 파이프라인 진행 상황 웹앱 실시간 표시
- Asana MCP 연동 (Phase 3)
- 오피스 UI 고도화 (에셋 부족)

### 신규 파일 (이번 세션)

```
src/lib/agents/
├── orchestrator.ts          ← 파이프라인 오케스트레이터
├── project_reader.ts        ← 프로젝트 폴더 접근
├── inter_message.ts         ← 에이전트 간 메시지 CRUD
src/app/api/pipeline/
└── route.ts                 ← 파이프라인 API (POST/GET)
src/data/agents/
└── planner.yaml             ← 기획자 에이전트
sql/
└── 002_create_pipeline_tables.sql
```
